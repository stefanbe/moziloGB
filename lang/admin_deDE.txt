Datenbank im Wartungsmodus befindet.<br /><b>Rot</b>, sobald sich mindestens eine Datenbank im Wartungsmodus befindet.<br />Zusätzlich werden die Datenbanknamen, die sich im Wartungsmodus befinden, enthalten.<br />Andere Zeichen werden beim Anlegen automatisch entfernt.<br />Die Datenbank darf noch nicht existieren.</p><h3 style="margin-bottom:0;">Altes Gästebuch importieren</h3><div style="padding-left:1.5em;margin-top:.3em;">Einfach die Gästebuch Datei in den <b>moziloGB/import/</b> Ordner kopieren.<br />Folgende Gästebuch Formate werden Unterstützt:<ul  style="margin-top:0;margin-bottom:0;"><li>moziloGB oder moziloGB Plugin für 1.12 (gb.txt)</li><li>www.onsite.org Gästebuch Script (gb.dat.php)</li></ul>Die importierte Datei wird automatisch aus dem <b>import</b> Ordner gelöscht.<br />Tip wenns den <b>import</b> ordner nicht gibt einfach erstellen.</div><h3 style="margin-bottom:0;">Aktive Datenbank und Sicherungskopien löschen</h3><p style="padding-left:1.5em;margin-top:.3em;">Die Datenbank mit all ihren Sicherungsdateien wird <b>unwiderruflich gelöscht</b>.</p><h3 style="margin-bottom:0;">Sicherungskopie</h3><p style="padding-left:1.5em;margin-top:.3em;"><b>Anlegen</b>: Es wird eine Sicherung von der aktuellen Datenbank angelegt - mit dem Datum und Zeit als Name.<br /><b>Wiederherstellen</b>: Die ausgewählte Sicherung wird zu einer "normalen" Datenbank und die aktuelle Datenbank wird als Sicherung angelegt.<br /><b>Löschen</b>: Die Ausgewählte Sicherung wird <b>unwiderruflich gelöscht</b>.</p><h3 style="margin-bottom:0;">Kommentare speichern</h3><p style="padding-left:1.5em;margin-top:.3em;">Es werden alle geänderten Kommentare gespeichert.</p><h3 style="margin-bottom:0;">Markierte Einträge löschen</h3><p style="padding-left:1.5em;margin-top:.3em;">Alle ausgewählten Einträge werden <b>unwiderruflich gelöscht</b>.<br /><b>Achtung</b>: wenn ein ausgewählter Eintrag weitere Einträge enthält, werden diese mitgelöscht.</p><h3 style="margin-bottom:0;">Datenbank Einstellungen:</h3><p style="padding-left:1.5em;margin-top:.3em;"><b>Antworten auf Eintrag zulassen</b>: Es kann auf einen Eintrag geantwortet werden, die Antwort gehört dann zu dem Eintrag.<br /><b>Kommentare für Eintrag zulassen</b>: Der Admin kann einen Eintrag mit einem Kommentar versehen.<br /><b>nur Lesen Modus</b>: Es gibt keine Möglichkeit mehr, neue Einträge zu erstellen.</p><h3 style="margin-bottom:0;">Eintrag verschieben</h3><p style="padding-left:1.5em;margin-top:.3em;">Sortiert wird grundsätzlich nach dem Erstellungsdatum, die neue Positionsnummer wird anhand dieser Sortierung vergeben.<br />Einträge der ersten Ebene können nur unter einen anderen Eintrag verschoben werden.<br />Um einen unteren Eintrag in die erste Ebene zu verschieben, einfach das Feld <b>unter den Eintrag (Pos.)</b> leerlassen.</p>
